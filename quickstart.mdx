## Prerequisites

Before you begin, ensure you have the following installed:

- **Node.js** (version 20 or higher)
- **npm**, **yarn**, or **pnpm** package manager
- **MongoDB** database (local or cloud instance like MongoDB Atlas)
- **Google OAuth Credentials** (for authentication)

<Note>
  If you don't have Google OAuth credentials, you'll need to create them in the [Google Cloud Console](https://console.cloud.google.com/).
</Note>

## Quick Setup

<Steps>
  <Step title="Clone the Repository">
    Clone the TeamUp repository to your local machine:
    
    ```bash
    git clone https://github.com/abz4375/TeamUp.git
    cd TeamUp
    ```
  </Step>

  <Step title="Install Dependencies">
    Install the required dependencies using your preferred package manager:
    
    ```bash
    npm install
    ```
  </Step>

  <Step title="Configure Environment Variables">
    Create a `.env.local` file in the root directory and add your environment variables:
    
    ```env
    MONGODB_URI=your_mongodb_connection_string
    AUTH_GOOGLE_ID=your_google_client_id
    AUTH_GOOGLE_SECRET=your_google_client_secret
    ```
    
    <Note>
      Replace the placeholder values with your actual MongoDB URI and Google OAuth credentials.
    </Note>
    
    **Get your MongoDB URI:**
    - For MongoDB Atlas: Use the connection string from your cluster dashboard
    - For local MongoDB: Use `mongodb://localhost:27017/teamup`
    
    **Get Google OAuth credentials:**
    1. Go to [Google Cloud Console](https://console.cloud.google.com/)
    2. Create a new project or select an existing one
    3. Enable the Google+ API
    4. Create OAuth 2.0 credentials
    5. Add `http://localhost:3000/api/auth/callback/google` to authorized redirect URIs
  </Step>

  <Step title="Run the Development Server">
    Start the development server:
    
    ```bash
    npm run dev
    ```
    
    Open [http://localhost:3000](http://localhost:3000) in your browser.
  </Step>

  <Step title="Sign In with Google">
    Click the "Sign In" button and authenticate using your Google account.
    
    <Note>
      Make sure you're using a Google account that you want to associate with your TeamUp profile.
    </Note>
  </Step>

  <Step title="Create Your First Project">
    Once signed in:
    
    1. Click on **"Create Project"** or **"New Project"**
    2. Enter a project name and description (supports Markdown)
    3. Click **"Create"** to initialize your project
    4. Start inviting team members and creating tasks!
  </Step>
</Steps>

## Troubleshooting

<AccordionGroup>
  <Accordion title="MongoDB connection error">
    If you're getting a MongoDB connection error:
    
    - Verify your `MONGODB_URI` is correct
    - Ensure MongoDB is running (for local instances)
    - Check network access settings (for MongoDB Atlas)
    - Make sure your IP address is whitelisted in MongoDB Atlas
  </Accordion>
  
  <Accordion title="Google OAuth not working">
    If Google OAuth authentication fails:
    
    - Verify `AUTH_GOOGLE_ID` and `AUTH_GOOGLE_SECRET` are correct
    - Ensure the redirect URI is added to your Google OAuth app settings
    - Check that the Google+ API is enabled in your Google Cloud project
  </Accordion>
  
  <Accordion title="Port 3000 already in use">
    If port 3000 is already in use:
    
    ```bash
    # Run on a different port
    PORT=3001 npm run dev
    ```
  </Accordion>
</AccordionGroup>
